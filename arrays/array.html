Array Bidimensional

No livro diz que Javascript aceita somente matriz unidimensional, porém eu consigo sim colocar um array dentro de outro array.

Em javascript o funcionamento da matriz é diferente. Em python era só botar a lista de linha que se quer percorrer e depois entrar dentro dessa linha e percorrê-la. Aqui, é necessário colocar todas as listas dentro de uma única lista, depois percorrê-la usando length, após isso, para percorrer as colunas dentro da linha, tem que especificar a posição de i(linha) que será percorrida.
